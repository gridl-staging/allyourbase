# Changelog

## 0.1.0

- Initial Dart SDK release.
- Auth parity: register/login/me/refresh/logout/deleteAccount/password reset/email verification/OAuth redirect.
- Records parity: list/get/create/update/delete/batch with query param support.
- Realtime parity: SSE parser, subscriptions, reconnect strategy.
- Storage parity: multipart upload/list/downloadUrl/delete/signed URL.
- RPC parity: `rpc()` function invocation.
- API key auth parity: `setApiKey` / `clearApiKey`.
- Push parity: register/list/revoke device tokens.
- Integration profile and full unit/contract test coverage.
